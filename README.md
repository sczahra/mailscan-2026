# MailScan 2026

MailScan 2026 is a local-first desktop utility for reviewing scanned mail, OCR PDFs, and financial-document exports.

## Safety model

This repository contains source code only.

Do not commit:

- scanned documents
- OCR PDFs
- financial spreadsheets
- exported bill data
- logs with private paths/details
- local settings
- old abandoned build folders

MailScan is intended to keep private documents local on the user's PC.
