# MailScan 2026 Design Notes

MailScan 2026 is manual-first.

Default rules:
- Never move source scans automatically.
- Never delete source scans.
- Keep scanned/OCR/private documents local.
- Repository contains source code only.
- Scan and OCR features should be explicit user actions.

Visual style:
- Vintage Mac utility influence.
- Soft gray panels.
- Compact toolbar.
- Table-centric document review.
- Clear disabled states for unfinished features.
